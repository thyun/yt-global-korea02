# 외국인의 한국 생활 — YouTube 동영상 제작 에이전트
다음 링크와 같은 영상을 계속 만드는 에이전트들을 수행하고 싶어
https://www.youtube.com/watch?v=H2KRGy6O0_0
다음 가이드를 참고해서 현재 프로젝트 디렉토리 구성해서 에이전트 수행 요청하면 수행가능하게 구성해 줘.

- 타겟 시청자들은 30대 이상 한국인들이야.
- Copilot 니가 에이전트가 되어서 각 단계들을 수행할 거야.
  각 단계별 에이전트 역할을 부여해서 수행해야 해.
- 에이전트 파이프라인을 구성해 주고 산출물에 대해서도 정의해 줘.
  각 단계별 입력물, 출력물들을 정의해 주고 단계별 디렉토리 구성도 해 줘.
- Copilot 니가 직접 수행할 수 있는 단계들은 직접 수행해 주고
  Youtube 검색 등을 위해 프로그램을 작성해서 처리해야 효율적인 것들이 있다면 프로그렘을 작성해서 수행해 줘.
